Streamline WooCommerce order management with seamless quick navigation buttons in the admin order edit screen.

== Description ==

**Quick Order Navigation for WooCommerce** is an essential tool for enhancing the order management experience within the WooCommerce dashboard. Designed to simplify workflow, this plugin adds intuitive 'Next' and 'Previous' navigation buttons directly on the WooCommerce order edit screen, making order processing faster and more efficient.

**Effortless Navigation:** Seamlessly move between orders without returning to the order list, saving time and clicks.

**Status-based Navigation:** (If applicable) Navigate through orders based on their statuses, bringing further efficiency to your workflow.

**HPOS Compatibility:** Fully compatible with WooCommerce's High Performance Order Storage, ensuring robust performance even in large-scale eCommerce operations.

Perfect for WooCommerce store administrators and managers, **Quick Order Navigation for WooCommerce** streamlines your order processing, allowing you to focus more on growing your business and less on administrative tasks.


== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The order navigation buttons will automatically appear on the WooCommerce order edit screen.
